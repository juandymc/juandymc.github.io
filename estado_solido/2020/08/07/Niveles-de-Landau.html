Gas de electrones en dos dimensiones

\[\begin{equation*}H=\frac{\pi^{2}}{2 m}=\frac{(\vec{p}-q \vec{A})^{2}}{2 m}\end{equation*}\]

dado que $\vec{B}=\vec{\nabla} \times \vec{A}$ si tomamos $\vec{A}=B_{o} x \hat{y}$ para que $\vec{B}=B_{o} \hat{z}$ podemos reescribir el hamiltoniano como

\[\begin{equation*}H=\frac{p^{2}}{2 m}-\frac{q B_{o} p_{y}}{m} x+\frac{q^{2} B_{o}^{2}}{2 m} x^{2}\end{equation*}\]

la componente $y$ conmuta con el hamiltoniano[] ¿?REFERENCIA A ALGÚN LIBRO DE CUANTICA DONDE SE VEA EXPLICITAMENTE LO QUE SUCEDE AL CONMUTAR … por tanto

\[\begin{equation*}\left[H, p_{y}\right]=0 \Rightarrow \psi(x, y)=e^{i k_{y} y} u(x)\end{equation*}\]

de modo que operando [1]

\[\begin{equation*}\begin{array}{r} H \psi_{p_{y}}=E\left(p_{y}\right) \psi_{p_{y}} \Rightarrow \\ \left(\frac{p_{x}^{2}}{2 m}+\frac{p_{y}^{2}}{2 m}-\frac{q B_{o} p_{y}}{m} x+\frac{q^{2} B_{o}^{2}}{2 m} x^{2}\right) \phi_{p_{y}}=E\left(p_{y}\right) \phi_{p_{y}} \Rightarrow \\ \left(\frac{p_{x}^{2}}{2 m}+\frac{\left(q B_{o}\right)^{2}}{2 m}\left(\frac{\hbar^{2} k_{y}^{2}}{\left(q B_{o}\right)^{2}}-2 \frac{\hbar k_{y}}{q B_{o}} x+x^{2}\right)\right) \phi_{p_{y}}=E\left(p_{y}\right) \phi_{p_{y}} \end{array}\end{equation*}\]
de modo que definiendo $\ell_{B}^{2} \equiv \frac{\hbar}{q B}$, $\omega_{c} \equiv \frac{\left q B_{o}\right }{m}$ y completando cuadrado
\[\begin{equation*}\frac{1}{2 m}\left(p_{x}^{2}+m^{2} \omega_{c}^{2}\left(x-k_{y} \ell_{B}^{2}\right)^{2}\right) \phi_{p_{y}}=E\left(p_{y}\right) \phi_{p_{y}}\end{equation*}\]

de modo que tenemos un oscilador armónico unidimensional con niveles

\[\begin{equation*}E_{n}=\hbar \omega_{c}\left(n+\frac{1}{2}\right)\end{equation*}\]

y con autoestados

\[\begin{equation*}\psi_{n, p_{y}}=e^{i k_{y} y} H_{n}\left(x-k_{y} \ell_{B}^{2}\right) e^{-\frac{\left(x-k_{y} \ell_{B}^{2}\right)^{2}}{4 \ell_{B}^{2}}}\end{equation*}\]

donde los $H_n$ son polinomios de hermite. Los niveles de energía de este oscilador en particular son llamadas Landau levels. Hay muchos estados cuánticos para cada Landau level, esto es, dado $n$, cada $p_y$ se corresponde a un autoestado con la misma energía.

El efecto de aplicar campo magnético se puede ver a continuación

La densidad de estados, constante en el caso $B=0$ se convierte en discreta al aplicar $B>0$.

Degeneración de los niveles de Landau

Tenemos condiciones de contorno periódicas luego $\Psi(x+L, y+L)=\Psi(x, y)$ [2, p. 138]

\[\begin{equation*}k_{y}=\frac{2 \pi n_{y}}{L_y}, n_{y}=0,\pm 1,\pm 2,\end{equation*}\]

pero también se cumple que

\[\begin{equation*}k_{y}=\frac{|q| B_0 x_{0}}{\hbar}\end{equation*}\]

de modo que igualando

\[\begin{equation*}x_{0}=\frac{2 \pi \hbar n_y}{|q| B_0 L_{y}} \quad \text { si } \quad 0 \leq x_{0} \leq L_{x}\end{equation*}\]

o equivalentemente

\[\begin{equation*}0 \leq x_{0} \leq L_{x} \Rightarrow 0 \leq \frac{2 \pi \hbar m}{|q| B_0 L_{y}} \leq L_{x}\end{equation*}\]

de este modo el número de electrones que puede ocupar un nivel de Landau es

\(\begin{equation*}0\leq n_y \leq \frac{|q| B_0 L_{x} L_{y}}{2 \pi \hbar}\end{equation*}\) de modo que el número máximo de electrones que puede ocupar un nivel de Landau sería

\[\begin{equation*}n_{y,\max }=\frac{|q| B_0 L_{x} L_{y}}{2 \pi \hbar}=N_{L}\end{equation*}\]

se suele usar la definición de longitud característica $l_B$ para simplificar

\[\begin{equation*}l_{B}=\sqrt{\frac{\hbar}{|q| B_0}} \Rightarrow N_{L}=\frac{L_{x} L_{y}}{2 \pi l_{B}^{2}}\end{equation*}\]

aunque también podemos interpretarlo desde el punto de vista de flujo magnético

\[\begin{equation*}N_{L}=\frac{|q| B_0 L_{x} L_{y}}{2 \pi \hbar}=\frac{B_0 S}{\frac{h}{|q|}}=\frac{\Phi}{\Phi_{0}}\end{equation*}\]

El número de estados cuánticos entre dos niveles de Landau consecutivos sería

\[\begin{equation*}\pi\left(k_{n+1}^{2}-k_{n}^{2}\right) \frac{L_{x} L_{y}}{(2 \pi)^{2}}=\pi \frac{2 m}{\hbar^{2}} \hbar \omega_{C} \frac{L_{x} L_{y}}{(2 \pi)^{2}}=\frac{|q| B_0 L_{x} L_{y}}{2 \pi \hbar}=N_{L}\end{equation*}\]
